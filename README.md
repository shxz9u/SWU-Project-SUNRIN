# 주제
라즈베리파이-클라우드 연계형 IoT 취약점 분석 시스템 연구: LLM 기반 분석과 Fuzzing 기법의 비교

# 요약
제한된 성능을 가진 IoT 환경에서 라즈베리파이를 보안 분석 게이트웨이로 활용하고, 라즈베리파이와 IoT 기기 간의 다양한 연결 방식을 어떻게 수집·분석할 수 있는지 연구한다. 라즈베리파이 자체의 연산 한계를 고려해 일부 분석은 외부 서버나 클라우드 LLM에 위임하는 하이브리드 구조를 설계한다. 이후 LLM 기반 취약점 분석 결과와 Fuzzing 기반 탐지 결과를 탐지율, 분석 시간, 리소스 사용량, 오탐률 기준으로 비교한다.
